# LANEye Architecture

## Components

- **Scanner**: ARP/ICMP network scanning
- **Backend**: FastAPI REST API
- **Database**: SQLite/PostgreSQL
- **Frontend**: React web GUI
- **Exporters**: ELK and Grafana integration
- **Notifications**: Multi-channel alerts
